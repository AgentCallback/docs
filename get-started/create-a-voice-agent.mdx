---
title: Create a voice agent
description: Build an inbound or outbound voice agent and run its first test.
---

Create a voice agent from the dashboard to define how AgentCallback handles phone conversations.

## Start a new agent

1. In the dashboard sidebar, select **Voice Agent**.
2. Select **Create New Script**.
3. Choose the call direction:
   - **Outbound** for calls your agent makes to customers.
   - **Inbound** for calls customers make to you.
4. Choose how to begin:
   - **AI Script Builder** to describe the call or paste an existing script. AgentCallback uses your input to create a starting configuration.
   - **Blank Script** to configure every detail yourself.

## Configure and save the agent

Complete the agent form with the information your agent needs to run the call. Give it a clear name and description, then set its instructions and any applicable voice, tools, integrations, or knowledge settings.

When you are ready, select the save action to create the agent. AgentCallback opens the new agent and starts its default initial test.

<Tip>
Use the AI Script Builder when you have a call goal or existing script but want a fast starting point. Use a blank script when you already know the exact configuration you need.
</Tip>

## Review the first test

1. Open the agent's **Tests** tab after it is created.
2. Review the automatic initial test and its results.
3. Adjust the agent configuration if the conversation does not meet your expectations, then test again.

## Advanced option: Pathways

The **Advanced** section includes **Pathways** (beta), a visual flow editor for calls that need fixed step order, tool gating, or escalation subflows. For most new agents, start with the AI Script Builder or a blank script.

## Next steps

After your agent behaves as expected, configure the phone number and campaign you will use to place or receive calls.
